Pyxie’s Grad Plan

This is a Godot platformer built from scratch for Module 1 - Module 2: Assignment - Individual Project #1

It recreates the first level layout of Super Mario Bros. 1-1 using original art and a new storyline.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

How to Run

- Open Godot 4.x

- Import the project using project.godot

- Click Play — the entry point is StartMenu.tscn

Important Note

If you click Play and the menu or story arc does not appear (and the game drops you straight into the gameplay loop), you’ll need to adjust the project settings:

- In the Godot editor, go to Project → Project Settings → Application → Run

- Set the Main Scene to StartMenu.tscn

After this change, the game will start correctly with the menu and story arc as intended.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Controls:

  - Move: Arrow keys or A/D

  - Jump: Spacebar

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Storyline: 
  
  - You play as Pyxie, a flame wisp caught in a storm with his smaller friends. After shielding them under a leaf, the rain stops and the world is now filled with water wisps. Pyxie sets out alone to find shelter   in a distant cave so his friends can stay safe.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Art Assets: 
  
  - All sprites and animations were created 100% by Alex Johnston in Aseprite. Only the background photo and sound effects were externally sourced.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Features:

  - Custom TileMap layout

  - Player movement, jump, stomp, bounce

  - Breakable boxes with random pickups

  - Bone pickup system

  - HUD showing bone count

  - Intro and ending cutscenes
